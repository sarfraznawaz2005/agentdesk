## Critical

The files in this `docs` folder are **design docs, plans, and PRDs** — some are
historical (plans/proposals kept for context) and may not reflect the current
code. Each doc that has drifted now carries a status banner at the top.

For the **current, verified** state of the system, use the project wiki:

- `project-wiki/index.md` — knowledge-base catalog (start here)
- `project-wiki/overview.md` — architecture narrative
- `project-wiki/reference/directory-map.md` — verified "where does X live?" map

When in doubt, the **code is the source of truth** — check it for the relevant task.
